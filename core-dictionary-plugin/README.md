Описание плагина генератора
===========================
Назанчение плагина - генерировать модуль для доступа к модели данных 
## Основное в данном плагине это потребители словарной системы:

  - FileConfigConsumer - сохраняет xml описание метамодели данных
  - FileClassConsumer - сохраняет calss описание метамодели данных
  - FileAspectConsumer - сохраняет описание аспектной модели 
  
## Шаблоны генерации модуля включают в себя:

  - ModuleDAO.vm -  основной класс DAO
  - persistence.vm - основной файл конфигурации orm
  - hbmxml.vm - файл сборщика конфигурации метаданных
